Hi! This is a set of small JavaScript projects. All the projects are collected in one place for convenience. Enjoy watching!

  • Photo finder

    Search applications, browse images by keyword. Use Pixabay's public API as the backend. The app creates an image gallery and has pagination.

  • Country search

    Search applications for data about a country by its partial or full name. Rest Countries public API is used. Field filtering and error handling are done.

  • Slider with live text

    Adaptive slider on native JS. Also designed live text animation using JS.

  • React feedback statistics

    The application is created using React, and designed in a different repository. The app counts the total number of collected reviews and the percentage of positive reviews.

  • Stopwatch

    The online stopwatch counts the time to the second that passes after you click the "Start" button.

  • Timer

    A timer script that counts down to a certain date. Uses the flatpickr library to allow the user to cross-browser select the target date and time.

  • Videoplayer with local Storage

    A script will save the current time of video playback to local storage and, when the page reloads, continue to play video from that time.

  • Gallery

    Gallery created dynamically from JS with using grids. The data for gallery is taken from json.

  • Local Storage Form

    The script will save field values to local storage when user writes something. When the page reloads, the field values will be automatically loaded from the local storage.

  • Color Switcher

    Changes the color of the background and text elements to a random value once a second

  • Promises Generator

    A script that creates promises as many times and with as much delay as entered in the field when the form is submited.

  • Box creator

    Creates collections of the written number of elements. Each element is 10px wider and higher than the previous one. Each element gets a random color.
